# Portfolio

**index.html:**
The index.html file is the main structure of the portfolio website. It includes a header, a navigation bar for different sections, and content sections such as Home, About, Projects, and Contact. The Home section contains an interactive hobby list where users can add and delete benefits dynamically. The Projects section showcases key projects with links, while the Contact section provides a form for user messages. 

**style.css:**
The style.css file manages the visual presentation of the website. It defines fonts, colors, spacing, layout alignment, and responsiveness. It also styles navigation links, project cards, form elements, buttons, hover effects, and ensures proper alignment of the hobby list with delete buttons. Media queries are used to make the website responsive across different screen sizes.

**script.js:**
The script.js file adds interactivity to the website. It enables users to add and delete items in the hobby list dynamically. It also handles contact form submission by displaying a confirmation alert and preventing page reload. Additionally, it displays the current date and time in the footer, updating automatically every second.
